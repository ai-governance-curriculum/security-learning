# AI Infrastructure Security Engineer Curriculum Overview

This repository provides lectures, exercises, and projects to help learners meet the expectations outlined in the role requirements. Use the mapping below to connect job requirements with specific modules and hands-on deliverables.

## Requirement-to-Module Mapping

| Job Requirement | Curriculum Coverage |
| --- | --- |
| Design and implement security architecture for AI/ML systems | Module 101 - ML Security Foundations<br>Module 102 - Zero-Trust Architecture<br>Project 01 - Zero-Trust ML Platform |
| Protect ML models from adversarial attacks and data poisoning | Module 106 - Adversarial ML<br>Module 112 - Advanced Topics<br>Project 03 - Adversarial Defense Lab |
| Implement compliance frameworks for AI systems (GDPR, HIPAA, SOC2) | Module 107 - Compliance & Governance<br>Module 110 - Supply Chain Security<br>Project 02 - Compliance Automation Playbook |
| Conduct security assessments and penetration testing of ML infrastructure | Module 108 - Runtime Security<br>Module 109 - Policy as Code<br>Project 04 - Secure ML CI/CD Pipeline |
| Design and enforce data privacy controls for training and inference data | Module 103 - Cryptography for ML<br>Module 105 - Secrets Management<br>Project 05 - Security Operations Runbook |
| Implement model security including input validation and output filtering | Module 106 - Adversarial ML<br>Module 108 - Runtime Security<br>Project 03 - Adversarial Defense Lab |
| Build audit logging and compliance monitoring systems for AI operations | Module 107 - Compliance & Governance<br>Module 109 - Policy as Code<br>Project 05 - Security Operations Runbook |
| Manage secrets, credentials, and access controls for ML infrastructure | Module 105 - Secrets Management<br>Module 102 - Zero-Trust Architecture<br>Project 04 - Secure ML CI/CD Pipeline |
| Implement network security for ML systems (zero-trust architecture) | Module 102 - Zero-Trust Architecture<br>Module 104 - Network Security<br>Project 01 - Zero-Trust ML Platform |
| Develop incident response procedures for AI security events | Module 111 - Security Operations<br>Module 107 - Compliance & Governance<br>Project 05 - Security Operations Runbook |
